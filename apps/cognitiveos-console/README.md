# CognitiveOS Console

面向 CognitiveOS 的跨平台可视化操作台，规划覆盖 Windows、Linux、macOS 桌面端，以及 iOS、Android 移动操作端。

- 默认入口：Agent Shell 对话界面
- 目标用户：终端用户、租户管理员、平台运维、安全审计与审批人员
- 规划能力：Agent 安装与管理、执行监督、记忆与知识图谱、用户与权限、多 Agent 协作、任务、审批、审计和可观测性
- 当前状态：`planned`；本目录只预留产品设计边界，不表示实现已提供、测试已执行或 Profile 已符合

产品设计文档生成目标：`PRODUCT-DESIGN.md`。技术栈、平台源码拆分和发布目录应在产品设计与 ADR 明确后再建立。
