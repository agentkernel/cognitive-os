# 模板 — Source Record

> 每条来源记录必须自包含。复制到相应 `../sources/*-ledger.md`。

## 记录字段

- 标题（页面/文档标题）：
- 完整 URL：
- 查询日：
- release / version / commit（如适用）：
- OS / build / SDK 适用范围（如适用）：
- 事实（只写来源明确陈述的内容）：
- 推论（明确标注为推论，与事实分离）：
- 未决（需进一步验证或真机 PoC）：
- 产品影响：
- 可借鉴 / 不可复制（如为竞品/许可来源）：

## 规则

- 事实与推论严格分离；不得把推论写成事实。
- 不得把营销文案当作实现事实；文档与源码冲突时登记为漂移。
- 许可/条款记录不构成法律结论；复用前须过法务 gate。
