# 模板 — Threat Record

> 复制到 [../security/threat-model.md](../security/threat-model.md)。ID 使用 `CONSOLE-AGENTHUB-V1-TM-*`。

## `CONSOLE-AGENTHUB-V1-TM-XXX` <威胁标题>

- asset：受保护资产。
- attacker：攻击者能力/动机。
- entry：攻击入口/接口。
- boundary：涉及的信任边界编号（见威胁模型 §1）。
- prevention：设计意图的预防措施（不代表已实现）。
- detection：检测手段。
- failure mode：预防失败时的后果。
- recovery：恢复/缓解路径。
- owner：负责子系统/车道。
- oracle：判定预防有效的未来测试（当前 not-run）。
- evidence：证据状态（当前 none）。
